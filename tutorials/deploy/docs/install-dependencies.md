# 安装依赖项

本阶段将完成 Vibe Coding 开发环境所需的基础依赖项安装，涵盖以下六个组件：

1. **PowerShell 7** — 跨平台任务自动化终端
2. **Windows Terminal** — 现代化终端应用程序
3. **Visual Studio Code** — 跨平台代码编辑器
4. **Python** — Python 运行时
5. **Node.js** — JavaScript 运行时
6. **pnpm** — 高效的 Node.js 包管理器
7. **Git** — 分布式版本控制系统

每节均配有详细的安装步骤截图和验证方法，确保零基础用户也能顺利完成。
